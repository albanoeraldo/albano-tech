# Albano Tech

Landing page desenvolvida para divulgação dos serviços da **Albano Tech**, com foco em captação de clientes para assistência técnica, manutenção de computadores, formatação, limpeza interna, upgrades e atendimento via WhatsApp.

A página foi criada como uma apresentação simples, direta e responsiva, com chamada para ação integrada ao WhatsApp, permitindo que o cliente entre em contato rapidamente para solicitar orçamento ou atendimento.

---

## Acesse o projeto

Projeto publicado via GitHub Pages:

https://albanoeraldo.github.io/albano-tech/

---

## Objetivo do projeto

O objetivo da página é apresentar os principais serviços da Albano Tech de forma profissional e facilitar o contato com clientes interessados em manutenção de computadores e notebooks.

A página funciona como uma vitrine digital para serviços de assistência técnica, ajudando a transformar visitantes em contatos pelo WhatsApp.

---

## Funcionalidades

* Página única responsiva.
* Apresentação dos serviços principais.
* Chamada direta para orçamento via WhatsApp.

## Serviços apresentados

A página destaca serviços como:

* Formatação completa.
* Instalação do Windows.
* Instalação de drivers.
* Otimizações de desempenho.
* Limpeza interna.
* Troca de pasta térmica.
* Checagem de temperatura.
* Upgrade de computador ou notebook.
* Recomendação de custo-benefício para melhorias.

---

## Como funciona o atendimento

O fluxo apresentado na página é simples:

1. O cliente chama no WhatsApp.
2. Informa o modelo do computador/notebook e o problema.
3. Recebe orçamento e prazo antes do início do serviço.
4. O serviço é executado, testado e entregue com recomendações.

## Destaques do projeto

### Integração com WhatsApp

A página possui botões de chamada para ação que direcionam o cliente diretamente para o WhatsApp, facilitando o contato e a solicitação de orçamento.

### Design responsivo

O layout foi desenvolvido para funcionar em diferentes tamanhos de tela, incluindo desktop, tablet e celular.

### Página objetiva

## Como executar localmente

Clone o repositório:


Acesse a pasta:

```bash
cd albano-tech
```

Abra o arquivo `index.html` no navegador ou utilize a extensão **Live Server** no VS Code.
